# 설정 가이드

설정 페이지에서는 무인 공간 관리 시스템의 모든 자동화 및 기기 동작을 상세하게 설정할 수 있습니다.

---

## 1. 입실 모드 설정

![입실 모드 설정](/1_quickmenu_open_setting.png)

**퀵 메뉴**의 **입실 모드** 실행 시 동작할 기기들을 설정합니다.

### 동작 방식

| 상태 | 동작 |
|------|------|
| ✅ 활성화 | 해당 기기 **ON** |
| ⬜ 비활성화 | 해당 기기 **OFF** |

### 설정 가능 기기

- 냉난방기 1, 2
- 실링팬
- 조명
- 공기청정기
- 환풍기
- 로봇청소기
- TV

## 2. 퇴실 모드 설정

![퇴실 모드 설정](/2_quickmenu_close_setting.png)

**퀵 메뉴**의 **퇴실 모드** 실행 시 동작할 기기들을 설정합니다.

### 동작 방식

| 상태 | 동작 |
|------|------|
| ✅ 활성화 | 해당 기기 **ON 유지** |
| ⬜ 비활성화 | 해당 기기 **OFF** |

::: tip 퇴실 모드 활용
퇴실 시에도 계속 가동이 필요한 기기(예: 로봇청소기)는 활성화 상태로 설정하세요.
:::

## 3. 예약 준비 모드 설정

![예약 준비 모드 설정](/3_booking_prev_setting.png)

**예약 준비** 시간에 자동으로 동작할 기기들을 설정합니다.

### 동작 방식

| 상태 | 동작 |
|------|------|
| ✅ 활성화 | 해당 기기 **ON** |
| ⬜ 비활성화 | 해당 기기 **OFF** |

::: info 예약 준비 시간
예약 준비 모드는 `자동화 설정` > `예약 준비 시간`에 설정된 시간 전부터 자동으로 실행됩니다.
:::

**퇴실 청소** 시간에 자동으로 동작할 기기들을 설정합니다.

### 동작 방식

| 상태 | 동작 |
|------|------|
| ✅ 활성화 | 해당 기기 **ON** |

## 5. 계절별 적정 온도 설정

![계절별 적정 온도 설정](/5_season_temperature_setting.png)

계절에 맞는 적정 온도를 설정합니다. 설정된 온도를 기준으로 자동화 기기들이 동작합니다.

### 계절 구분

| 계절 | 기간 |
|------|------|
| **하절기** | 6월 ~ 8월 |
| **동절기** | 12월 ~ 2월 |
| **간절기** | 3월 ~ 5월, 9월 ~ 11월 |

### 설정 항목

| 항목 | 설명 | 권장값 |
|------|------|--------|
| **하절기 온도** | 여름철 냉방 목표 온도 | 24°C |
| **동절기 온도** | 겨울철 난방 목표 온도 | 28°C |
| **간절기 온도** | 봄/가을 목표 온도 | 26°C |

## 6. 영상방송 설정

![영상방송 설정](/6_broadcasting_setting.png)

입실/퇴실 시 알림 화면을 설정할 수 있습니다.

### 설정 항목

| 항목 | 설명 |
|------|------|
| **입실 알림 사용** | 입실 시 알림 화면 ON/OFF |
| **퇴실 알림 사용** | 퇴실 시 알림 화면 ON/OFF |
| **입실 화면** | 입실 시 표시할 레이아웃 선택 |
| **퇴실 화면** | 퇴실 시 표시할 레이아웃 선택 |
| **입실 화면 변경 시점** | 입실 후 X분 뒤 화면 변경 |
| **퇴실 화면 변경 시점** | 퇴실 전 X분 전 화면 변경 |

::: tip 레이아웃 관리
레이아웃은 Xibo CMS에서 미리 생성해두어야 선택 가능합니다.
:::

## 7. 자동화 설정

![자동화 설정](/7_booking_automation_setting.png)

예약 시간에 맞춰 동작하는 자동화 시간을 설정합니다.

### 설정 항목

| 항목 | 설명 | 기본값 |
|------|------|--------|
| **예약 준비 시간** | 예약 시작 X분 전 준비 모드 시작 | 40분 |
| **퇴실 청소 대기** | 예약 종료 X분 후 청소 모드 시작 | 5분 |
| **오픈 청소 시간** | 매일 오픈 전 자동 청소 시간 | 08:00 |
| **마감 청소 시간** | 매일 마감 후 자동 청소 시간 | 23:00 |

### 동작 흐름

```
[예약 준비 시간]          [예약 시간]          [퇴실 청소]
     ↓                      ↓                   ↓
  X분 전 시작 ──────→ 예약 시작~종료 ──────→ X분 후 30분간 청소
```

::: warning 오픈/마감 청소
오픈 청소 시간과 마감 청소 시간은 예약과 관계없이 **매일** 설정된 시간에 동작합니다.
:::

## 8. 지점 환경 기본 설정

![지점 환경 기본 설정](/8_space_temperature_setting.png)

고객 예약과 관계없이 해당 공간의 온도를 일정하게 유지하고 싶을 때 설정합니다.

### 설정 항목

| 항목 | 설명 |
|------|------|
| **지점 환경 설정 사용** | 기능 ON/OFF (ON 시 매일 동작) |
| **시작 시간** | 온도 관리 시작 시간 |
| **종료 시간** | 온도 관리 종료 시간 |
| **설정 온도** | 목표 온도 |
| **운전 모드** | 자동/냉방/난방 선택 |

### 동작 방식

1. **지점 환경 설정 사용**이 ON 되어 있어야 동작합니다.
2. 설정된 시간 동안 지속적으로 지점 온도를 체크합니다.
3. **설정 온도**와 현재 온도를 비교하여 냉난방기를 자동 가동합니다.

::: info 운전 모드
- **자동**: 현재 계절에 맞게 냉방/난방 자동 선택
- **냉방**: 강제 냉방 모드
- **난방**: 강제 난방 모드
:::

## 9. 자동 전원 끄기 설정

![자동 전원 끄기 설정](/9_all_close_setting.png)

**전체 기기**에 대한 마감 체크를 설정합니다.

### 설정 항목

| 항목 | 설명 |
|------|------|
| **자동 꺼짐 사용** | 기능 ON/OFF |
| **자동 꺼짐 1** | 첫 번째 마감 체크 시간 |
| **자동 꺼짐 2** | 두 번째 마감 체크 시간 |

### 동작 방식

1. 설정된 시간에 해당 장소의 모든 기기 상태를 체크합니다.
2. 가동 중인 기기가 있으면 자동으로 **OFF** 합니다.

::: tip 이중 체크
자동 꺼짐 1, 2를 다른 시간으로 설정하면 이중으로 마감 체크가 가능합니다.
예: 자동 꺼짐 1 = 22:00, 자동 꺼짐 2 = 24:00
:::

## 10. 설정 저장

![설정 저장](/10_save_button.png)

모든 설정 변경 후 반드시 **저장 버튼**을 클릭해야 합니다.

::: danger 중요
설정값을 입력하거나 변경한 후 **저장 버튼을 누르지 않으면** 설정이 저장되지 않으며, 의도한 대로 동작하지 않습니다.
:::

### 저장 확인

- 저장 성공 시 확인 메시지가 표시됩니다.
- 저장 후 페이지를 새로고침해도 설정값이 유지되는지 확인하세요.

## 설정 요약

| 구분 | 설정 항목 | 용도 |
|------|----------|------|
| **모드 설정** | 입실/퇴실/준비/청소 모드 | 각 상황별 기기 동작 정의 |
| **온도 설정** | 계절별 적정 온도 | 자동화 기준 온도 |
| **영상방송** | 입실/퇴실 알림 화면 | 디지털 사이니지 연동 |
| **자동화** | 시간 기반 자동화 | 예약 연동 자동 제어 |
| **환경 설정** | 지점 환경 기본 설정 | 상시 온도 관리 |
| **마감 설정** | 자동 전원 끄기 | 에너지 절약 및 안전 |
